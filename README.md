### Set up
- Install Node.js
- Install npm:
  ``` bash
  npm i
  ```
- Tạo MySQL container (app Docker):
  
  ![image](https://github.com/user-attachments/assets/47e2a766-d132-4502-a8a9-6dd6ca723ebb)

### Start the application: 
- ``` bash
  npm run dev
  ```
